# Project Name
This is an Android app that is meant as a lyric trivia game. A user will either select an artist or get one randomly selected, and is then redirected to the game where a user has to fill in a song lyric from the chosen artist.

## Usage
Once the app is launched, follow these steps to play:

1. Select Your Artist: Browse the list of available artists.
    Use the Search bar to find a specific favorite.
    Or, tap "Random Artist" if you're feeling adventurous!
2. Start the Game: You will be presented with a famous lyric snippet.
3. Fill in the Blanks: Type the missing words into the input field.
